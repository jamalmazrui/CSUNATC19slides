Present – Low-level API

ž  Supported in UWP (Universal Windows Platform)

ž  Applications need gazeInput capability in Package.appxmanifest

ž  User needs to approve applications use on first launch

ž  Found in namespace Windows.Devices.Input.Preview

ž  GazeDeviceWatcherPreview allows you test for presence of an eye tracker

ž  GazeInputSourcePreview provides information about gaze falling on application

 

<Capabilities>

<DeviceCapability Name="gazeInput" />

 

Present – Windows Toolkit

ž  Install NuGet package Microsoft.Toolkit.Uwp.Input.GazeInteraction

ž  Add gaze namespace to XAML and enable it on gaze driven pages

 

<Page xmlns:gaze="using:Microsoft.Toolkit.Uwp.Input.GazeInteraction"

    gaze:GazeInput.Interaction="Enabled">
