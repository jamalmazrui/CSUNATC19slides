Big Data

Big Data

More than a buzz term and certainly not a passing trend. Big Data identifies a new era under which the world’s economy, power structures, civil liberties and our basic quality of life is being transformed. Big Data is no different than Big Oil or Big Pharma.

Discipline: Application of data-driven science to large inter-connected data-sets for the purpose of decision making.

•         Data Mining and Feature Extraction (70%)

•         Modeling: Statistical Analysis, Visualization and Algorithmic Tuning (20%)

•         Deployment for Real Time and Perpetual Use (10%)

 

 

Big Data and ARC

•             Qualitative: Does the application or content comply with automated test criteria set forth in accessibility guidelines? (Pass/Fail)

•             Quantitative: A subjective measurement of the usability of an application and/or content from the perspective of the consumer. (Usability Scale of 1 to 10)

•             Semi-Quantitative: The extent to which applications and content comply with manual test criteria set forth in accessibility guidelines. (Pass/Fail with Usability Scale)

Machine Learning

Machine Learning

Machine Learning (ML) is a branch of AI which uses statistical models on large data sets to make predictions and decisions, without the need for classic computing algorithms.

•         Predictive Analysis: Analyze current and historical facts to make predictions about the future

•         Clustering (Cluster Analysis): The unsupervised task of grouping (labeling) objects in a data set based on observed feature similarity.

•         Classifier (Classification): A model which utilizes supervised learning to classify (label) an object based on its features.
