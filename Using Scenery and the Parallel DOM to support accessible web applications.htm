The View: Scenery

 Open source graphics library for HTML5, by PhET

 

 Scenery is a scene graph implementation

 Scene graph is a "tree" like data structure for what is displayed on screen  Items in the tree are called "Nodes" and have parentchild relations with other nodes in the graph

The View: Scenery

 "Nodes" have graphical

information

Image data

Shape with fill and stroke

Transformation

(translation, rotation, scale)

 

 Scenery converts scene graph into visual display

The View: Scenery

 Scenery supports HTML

Canvas, SVG, and WebGL

 All three of these renderers can be used in a single application

 

 Example: Scenery Playground

Accessibility in traditional Web Graphics

 Canvas:

Sub DOM inside canvas addHitRegion, removeHitRegion, drawFocusIfNeeded

 WebGL: Sub DOM inside Canvas

 SVG: Native markup (title, desc), ARIA enhanced SVG

 

Issues with these approaches

 Recommendation for each renderer is different

 Supporting each traditional approach triples cost  Experimental technologies have limited support and cannot be used in production code

The Parallel DOM

 Parallel HTML tree in addition to graphics

 

 Accessible representation for what is displayed

 Independent of other scenery graphics rendering

 Visually hidden, available for assistive technologies

 Leverages HTML and ARIA

for best of web accessibility

Conclusions
