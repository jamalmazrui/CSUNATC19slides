What is a Switch?

An external device that can be mapped to commands that allow people to select items, enter text, and use their mobile device.

Enabling Switch Access for Testing

Once you have a Bluetooth keyboard set up for the keyboard testing, it can also be mapped as an external switch for Switch Access 

On iOS

Settings > General > Accessibility > Switch Control

 

•                In Switch Control, tap on Switches.

•                “Full Screen” should be set up by default. To set up the keyboard as a switch tap “Add New Switch” to set it up.

•              Under “Source” select “External”

•              When asked to “Activate your external switch”, tap the key you want to assign.

•              Choose the action you want to assign to this key – for the first button we chose “Move to Next Item” to make it our “Next” switch

•                Repeat steps 3-6 to add a second switch.  On the second switch, choose “Select Item” instead of “Move to Next Item”.  (You might experiment with other options like “Tap” if your user can control more than 2 switches)

•                Once your switches are setup, return to the main Switch Control screen and enable Switch Control.   

On Android

Settings > Accessibility > Dexterity > Universal Switch

•              In the Switch Access settings, 

•              Tap Auto-scan.

•     If Auto-scan doesn't appear, tap Scanning method and select Linear scanning or Row-column.

•              At the top of the Auto-scan screen, turn on Auto-scan.

•     Optional: Adjust auto-scan settings, including auto-scan time, delay on first item, and number of scans.

•              Tap the Back button to return to Switch Access settings.

•              Select “Assign switches for scanning”. 

•              Tap Auto-scan.

•              When the dialog opens, press your switch.

•              Tap Save.
