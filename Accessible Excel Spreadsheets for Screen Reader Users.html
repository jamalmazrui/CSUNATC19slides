7 Steps to Check Excel Accessibility

Name Each Worksheet

Name is important

  • Name an object for easier navigation.
  • Give all sheet tabs, data table, charts/ images/ shapes/ SmartArt unique & meaningful names.
  • If one spreadsheet has multiple sets of data, one data table per spreadsheet is recommended.

Provide Help Text

Provide Help text in A1 and A2

  • A1: How to navigate the page with screen reader. Type “Press UP or DOWN ARROW in column A to read through the help text and descriptions.”
  • A2: What is the structure of the worksheet. Type number of tables and charts with their corresponding names and starting cells. Extra information, like data source, is also included here.

Hide text

  • Note: After writing the text, reduce the cell size and change the text color to match the background color. With these changes, screen readers can recognize the help text that will benefit screen reader users, and sighted users won’t be disturbed by this additional information.

Column A Is Useful

Left alignment is important

  • Screen readers start reading the information in cell A1. Empty column A can confuse screen reader users as they might think the worksheet is empty.
  • Align all the tables and charts to the left, starting in column A.

When not to left align?

  • Exception: sometimes when students need to interact with the data, it is recommended not to left align the tables and/or charts. This can interfere with the development of the activity.

Table 1: Setup

Provide description for each table

  • ln A3, provide info about a table including table name, the range of this table, number of columns and rows. You can also include any extra information when needed.
  • For more than one table, use the row above each table to provide the description.
  • Match the description with the background color.
  • Example: The “Frequency” table ranges from A4 to E2196. It has 5 columns and 2191 rows. This table is generated from the previous complete data.

Provide column headers

  • Screen reader will repeat the column header info as a guidance to read the table.
  • Insert a new row above the table and each header needs to be unique and meaningful.

No merged cells in the DATA part!

  • Select the merged cell,
  • go to Home tab and click “Merge & Center.”
  • right click, choose the “Format cells”, click the “Alignments”, uncheck the “Merge cells.”

No empty cells in the DATA part!

  • Check empty cells and add “No data.”
  • Change the text color to match the background, so only a screen reader will read it.

Table 2: Construction

Set table format

  • To ensure that a screen reader interprets your column headers as headers: Select the table.
  • Set it into “All Borders.”
  • In the Insert tab, choose Table to create table, and check “My table has headers” checkbox. Confirm the range is correct.

Set table name in table format

  • Choose a unique valid name.
  • Space is not allowed here.
  • The name should begin with a letter or an underscore.
  • With multiple worksheets, name the same table as TableName_in_Worksheet_Name.

Set table name in table format

  • Click the Table tab in the top menu and provide a valid name for the table.
  • Note: For a table with no headers, select the Header Row checkbox. Then, include a column name.

What if Table format does not work?

  • To format a table manually, select the top-left cell of the table, which is the column header.
  • Go to Formulas tab and select Name Manager (Windows) or Define Names (Mac).
  • Add a name of the table.

Set table style

  • Select any cell in the table.
  • In the Table tab, go to Design and select Table style (Windows) or Quick Style (Mac).
  • Choose a style with black and white color (e.g., “Table Style Medium 1.”)

Provide alt text to tables

  • Right-click a table.
  • Select Table and choose the Alternative Text.
  • Type a description and a title. You can reuse the description for the table (e.g., A3).

Add the ending

  • At the end of each table, add a cell with “End of table name table.”
  • At the end of the worksheet, add a cell with “End of worksheet.”
  • Make sure to match the text with the background color.

Provide Description for Graphs and Images

Provide a brief description of the object in the cell (column A).

  • Use one cell for one object to add the description.
  • Resize the row where you want to insert the object.
  • Change the text color to match the background.
  • Move the object into the cell.

Add Alternative Text Description

  • Right-click an object to get to Size & Properties.
  • Add a title and description. You can reuse the description of the object.
  • Description guide: Diagramcenter.org (http://diagramcenter.org/specific-guidelines-e.html)

Visual Description Example 1: Bar graph

The graph is a histogram that shows the average dollars spent on coffee per month. The horizontal x-axis shows dollars spent on coffee with bins ranging from 5 to 55 in increments of 5. The vertical y-axis shows the height of each bin, which represents frequency and ranges from 0 to 45 in increments of 5.

Visual Description Example 2: Scatter plot

The graph is a scatter plot with no title listed. The horizontal x-axis shows height in centimeters ranging from 130 to 180 in increments of ten. The vertical y-axis shows weight in kilograms ranging from 30 to 90 in increments of ten. The graph contains approximately 100 points that seem to be evenly spread along a linear trend that goes from the lower left corner, where weight and height are small, to the upper right corner, where both quantities are large.

Hide or Delete Unused Information

Hide extra columns and rows

Columns

  • Select the 1st column you want to hide.
  • Press Ctrl (Command) + shift + right arrow.
  • With your cursor hovering in the selected area, right-click and choose Hide.

Rows

  • Select the 1st row you want to hide.
  • Press Ctrl (Command) + shift + down arrow.
  • With your cursor hovering in the selected area, right-click and choose Hide.

What if you cannot hide rows and columns?

  • Check whether the selected region has any merged cells. Try unmerging them before hiding unused rows.
  • Try a different order of hiding columns and rows.

Unhide extra rows and columns in case

  • Select the last visible column.
  • Hover to the right side of the right edge of the column header. Your cursor should turn into a double cursor with arrows.
  • Click and drag to the right, exposing the very last column.
  • Select the new, final column, right-click and choose Unhide.

Delete any unused worksheets

  • Right-click on the tab of the unused worksheet and choose Delete.
  • Note: Be careful! This CANNOT be undone.
